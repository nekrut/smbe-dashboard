# Orbit Agent — SMBE Tester Dashboard

**Live:** https://nekrut.github.io/smbe-dashboard/

A public, **anonymized** dashboard for the **SMBE** testing group in the
[Orbit Agent](https://galaxyproject.org/tools/orbit/) beta — the Galaxy Project's AI co-scientist
for bioinformatics. It shows each tester's activity (bug reports filed through Orbit's in-app
Feedback button) and their API spend.

Testers appear by an anonymous ID (`SMBE1`…`SMBE13`) only — no names or emails. The page is
generated automatically and refreshes a few times a day.
